## Nasıl Kullanılır?

1. Önce bu klasördeki ilgili sprint dosyasını oku
2. Dosyada geçen kod yollarını projede aç ve karşılaştır
3. `docs/03-INTERN-TASKS.md` içindeki checkbox'ları kontrol et
4. Test senaryolarını uygulamada dene
5. Bir sonraki sprint'e geç
